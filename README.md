# Sokoban SDL2

This project is the result of my coursework (2018–2019).  
It’s a modified Sokoban game written in **C++** using the **[SDL2](https://www.libsdl.org/)** and **SDL2_ttf** libraries.  
This is my second project using SDL; my first one was written in **C** with **SDL 1.2**.  

The game was developed and run on Kubuntu 19.04.
